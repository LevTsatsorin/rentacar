PASOS PARA EJECUTAR:
1. composer install
2. cp .env.example .env
3. php artisan key:generate
4. Configurar DB_DATABASE, DB_USERNAME y DB_PASSWORD en .env
5. php artisan migrate:fresh --seed
6. php artisan storage:link
7. php artisan serve
